
Standalone application with third-party dependency

*****
Requirements
*****

Python version
########

Make sure you have python3 installed

.. code-block:: bash

    python3 --version

Poetry
########

Make sure you have `Poetry <https://python-poetry.org/>`_ installed

.. code-block:: bash

    poetry --version

*****
Run
*****

.. code-block:: bash

    poetry install
    poetry run currency.py

Sources
*****
